DA HARDCORE RULEZ:

  1. Battle style

    2. We’ll be rocking“Set” mode!“Shift” mode allows the player to get the advantage on every single opposing Pokemon.

  2. Encounters!

    3. Per standard nuzlocke rules, we will be using only the first Pokemon we encounter per route. Typically, this includes gift encounters.

    The route that a Pokemon counts for is defined by the location met within their Summary. There are exceptions to the rule that are game-specific, though this rule is typically hard-set.

  3. Level Caps!

    4. The ensuing gym leader’s highest leveled Pokemon is our level cap.

    Leveling up during the battle with the gym-leader is permitted

    If a Pokemon surpasses the current set level cap, it is still available for use outside of the gym battle, and not available for use against the gym leader.

  4. Fainted Pokemon

    5. If a Pokemon Faints, it must be placed in the death box.

    I think it’s fun at the end of the run to look back at all the fallen friends that put their heart into a successful run. Plus, releasing them hurts my feelings way too much.

  5. Duplication Clause!

    6. We may not catch two Pokemon of the same species within the same run!

    We may, however, continue to reroll the encounter until we find an encounter we have yet to find!

  6. Shiny Clause!

    7. We all love shinies, so if we find a random shiny it we can use it if we catch it, but it has to be random.We can't shiny hunt.

  7. Items in Battle

    8. Only HELD ITEMS are permitted! Just to make things a little spicier! If it can’t be held, it can’t be used!
